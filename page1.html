2. What is a variable?

Variables a used to store data. It is like a container that we use to store things at home.

In JavaScript we use var, let and const to define a variable.

2.1 Variable and scope

Let keyword in JavaScript is used to declare a block scoped variable. This means that the variable is only visible within the block in which it is declared
